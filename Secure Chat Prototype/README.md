# Quantum-Resistant Encrypted Chat
This quantum resistant prototype demonstrates 2-party encrypted messaging using Kyber, a post-cryptography encapsulation key, and Advanced Encryption Standard (AES). In this case, the 2 parties are Alice and Bob. One person sends a message then the program encrypts the message. Then the other party will decrypt the message using the secret shared key.


## Project Overview
- Implements 3 variants of Kyber to evaulate and compare the security and processing time.
- Implements AES to generate a 256-bit shared AES key.
